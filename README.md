# Interactive To-Do List Application

A web-based task management application built to master JavaScript DOM manipulation, event handling, and data persistence using `localStorage`.

## Features
- **Create**: Add new tasks to your list.
- **Read**: View all tasks dynamically.
- **Update**: Toggle task completion status.
- **Delete**: Remove tasks from the list.
- **Persistence**: Data remains saved across browser reloads using `window.localStorage`.
- **Filtering**: View 'All', 'Active', or 'Completed' tasks.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
